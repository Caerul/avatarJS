avatarJS
========

Drop image, crop, push


Requires
========

jQuery
Jcrop


Usage
========

You know how this works. Just pass in the AJAX options you want for submitting the cropped image

$(selector).avatar({
});


Options
========

{
	uploadURL:		"",
	uploadData:		{},
	uploadDone:		function(e){},
	uploadFail:		function(e){}
}
